# E-commerce-Sales-Dashboard

1. Overview

This project presents an interactive E-commerce Sales Dashboard built using Power BI. The main objective is to analyze online sales data and generate meaningful insights related to sales performance, customer behavior, and profit trends.
It demonstrates a complete data analysis workflow, including data preparation, modeling, and visualization.

2. Dataset

The project is based on two datasets:

Orders Data – Contains transaction details such as order amount and quantity
Details Data – Includes category, sub-category, payment mode, and customer information

Both datasets were initially in Excel format and imported into Power BI for further processing.

3. Data Modeling

. Created relationships between tables using common fields (e.g., Order ID)
. Ensured accurate data flow and consistency across visuals
. Enabled multi-dimensional analysis

 4. Key Metrics (KPIs)

The dashboard includes the following key performance indicators:

. Total Sales
. Total Quantity Sold
. Total Profit
. Average Sales Value

These KPIs provide a quick overview of business performance.

5. Features

. Interactive Quarter-wise filter (Q1–Q4)
. State filter for regional analysis
. Dynamic dashboard updates based on user selection

6. Visualizations

The dashboard includes multiple types of visualizations:

. Sales by State – Identifies top-performing regions
. Quantity by Category – Shows category-wise distribution
. Profit by Month – Displays monthly profit trends
. Profit by Sub-Category – Highlights most profitable products
. Sales by Customer – Shows top customers
. Quantity by Payment Mode – Displays customer payment preferences

7. Dashboard Design

. Clean and professional dark theme
. Well-structured layout for easy understanding
. KPIs placed at the top for quick insights
. Consistent color usage for better readability

8. Key Insights

. Clothing category contributes the highest sales
. Cash on Delivery (COD) is the most used payment method
. Maharashtra generates the highest sales
. Profit is highest in November
. Some months show low or negative profit, indicating improvement areas

9. Learnings

. Built interactive dashboards using Power BI
. Performed data modeling and table relationships
. Used DAX for creating measures and calculations
. Applied filters and slicers for dynamic analysis
. Designed user-friendly and insightful visualizations

10. Conclusion

This project showcases an end-to-end data analytics solution using Power BI. It highlights the ability to transform raw data into actionable insights and supports data-driven decision-making.
